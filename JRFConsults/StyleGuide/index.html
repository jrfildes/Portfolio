Fonts

Framing the site in a hand-drawn format, we'll be continuing that theme with handwritten/drawn styles.

The fonts used will be as below:

  1. Permanent Marker: for H1, H2, H3 
  2. Montserrat for P

Text Styles

H1

Text must be in Playfair Display format, font 35px

H2

Text must be in Permanent Marker format, font 28px

H3

Text must be in Permanent Marker format, font 20px

P

Text must be in Monserrat format, italic, font 15px
